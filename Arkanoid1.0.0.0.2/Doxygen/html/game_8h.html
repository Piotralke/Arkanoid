Funkcja game() jest funkcją zawierającą główną pętlę gry. W każdym przejściu pętli odświeżany jest ekran, są od nowa rysowane elementy o zmienionych pozycjach. Sprawdzana jest także ilość żyć oraz bloczków, które odpowiednio kończą grę w przypadku braku żyć oraz przechodzą do następnego poziomu w przypadku zbicia wszystkich bloczków.
